About Conference

Front-Trends is a 3-day conference based in Warsaw for web and mobile developers to discover the latest trends, technologies, and ideas in front-end development. For over 7 years, Front-Trends has brought together people from all over the world who are shaping the face of the internet and the future of human-technology interaction.

Una Kravets: The Power of CSS

We can make many Photoshop look-like effects just using css. Una showed us 10 effects written in css: modals, lightboxes, tooltips, scroll indicators, carousels ect... They don't need JS. You can have the same results using target, data attributes, siblings- and pseudo-selectors.

Jenna Zeigen: On How Your Brain is Conspiring Against You Making Good Software

Turns out, humans aren't particularly a good fit for programming. We are predictably irrational (Dan Ariely) - we employ tricks, patterns and shortcuts to think fast, but this has downsides for our judgment. We constantly make errors called cognitive biases. And if software is for people written by people, how do we make sure we don't carry these biases on to our code?

Kirupa Chinnathambi: Building a Progressive Web App. 1/3

Progressive web apps are apps that are:

  • secure (https)

  • fast (spa)

  • can sent push notifications

  • can appear in the home screen with an icon, a manifest, etc.

Kirupa Chinnathambi: Building a Progressive Web App. 2/3

PWA contains:

  • Web app

  • Manifest file - JSON file that provides information about your web app.

  • Service workers - JS file. It acts as a proxy that sits between your web app, the browser, and the network, when available.

Kirupa Chinnathambi: Building a Progressive Web App. 3/3

  1. PWAs are just web apps thay provide a great user experience.
  2. Use progressive enhancement so everyone can use them.
  3. PWAs work great both inside and outside the browser.
  4. Sometimes they're the right solution, sometimes not, and that's okay.
